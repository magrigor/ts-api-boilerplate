## Typescript Node.js Api Boilerplate

Clone the repository
Install dependencies
```
npm install
```
Modify files under ```src/```
Start the application with
```
npm start
```
Enjoy :)


### Start the application in prod env
```
npm run start:prod
```

### Build application
```
npm run dist
```

### Run linter
```
npm run lint
```

P.S Not supporting debugging :(
